# Jain Polymer Co. — Wire Cost Calculator

A premium, high-performance web application designed for industrial wire cost estimation. This tool provides precise calculations for conductor diameters, material weights, and total production pricing for Jain Polymer Co.

## ✨ Features

- **Industrial Calculation Engine:** Accurately calculates wire diameter and weight metrics based on conductor size and insulation thickness.
- **Real-time Cost Estimation:** Instantly computes copper and resin/tape costs using live market rates.
- **Premium UI/UX:** Features a modern, glassmorphic dark-mode interface with smooth animations and interactive elements.
- **Progressive Web App (PWA):** Can be installed on mobile devices and works offline via Service Workers.
- **Clean Start:** Designed for client interaction with empty initial fields and manual calculation triggers.

## 🧮 Calculation Logic

- **Diameter:** Calculates the overall insulated diameter based on square-millimeter conductor size.
- **Copper Weight:** Precision weight estimation in kg per 100 meters.
- **Resin / Tape Weight:** Specialized volume-based weight calculation for insulation layers.
- **Pricing:** Dedicated pricing logic for copper (kg-based) and resin (gram-based) materials.

## 🛠️ Technology Stack

- **Frontend:** Vanilla HTML5, CSS3 (Custom Design System), and JavaScript (ES6+).
- **Offline Support:** Service Worker API for caching and offline functionality.
- **PWA Manifest:** Complete mobile installation support.
- **Typography:** DM Sans and Space Mono via Google Fonts.

## 🚀 Deployment

The application is optimized for static hosting platforms like **Vercel** or GitHub Pages. Simply push to a repository and link it to your hosting provider.
